# Landing Hero Image

## Image File Required

Place the traditional Indian wedding couple hero image here with the filename:

**`landing_hero.jpg`**

## Image Specifications

- **Format:** JPG or PNG
- **Recommended Size:** 1080x1920px or similar portrait orientation
- **Aspect Ratio:** Portrait (vertical) preferred for mobile
- **File Name:** `landing_hero.jpg` (or `landing_hero.png`)

## Usage

The image is used as a full-screen background on the Landing/Welcome screen.

If the image file is not found, a fallback grey container with an icon will be displayed.
