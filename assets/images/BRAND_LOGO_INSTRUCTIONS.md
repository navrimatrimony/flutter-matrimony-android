# Brand Logo Image Placement

## Brand Name
**नवरी मिळे नवऱ्याला**

## Logo File Required

Place the full brand logo image (text + illustration) here with the filename:

**`brand_logo.png`**

## Logo Specifications

- **Format:** PNG (with transparency preferred)
- **Recommended Size:** 
  - For Landing screen: 400-600px width (will be scaled to height: 120px)
  - For Drawer: 200-300px width (will be scaled to height: 40px)
- **Aspect Ratio:** Horizontal/landscape preferred
- **File Name:** `brand_logo.png`

## Usage Locations

1. **Landing Screen:** Top center, height: 120px
2. **Drawer Header:** Left side with brand name, height: 40px

## Fallback

If the logo file is not found, the app will display:
- Landing screen: Heart icon + brand name text
- Drawer: Heart icon only
